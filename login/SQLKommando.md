# SQL Kommandoer

Hei her er informasjon om Login Systemet til mitt nettsted. 


SQL Kommandoer 

Innloging : 

sudo mysql -u brukernavn -p (passord)


Bruk av database : 

use database 
Example: use LoginSystem

viewing tables :

SELECT * FROM (table);
Example: SELECT * FROM user;

viewing database description :

DESCRIBE (table);
Example: DESCRIBE user;
